Importera produkter (Excel)

I Bokfri Administration kan du importera enstaka produkter eller ett helt produktregister från en Excel-fil. Du sätter rubriknamn på första raden för varje kolumn i Excel-filen och Bokfri Administration läser av rubriknamnet och lägger kolumnens data som finns sparad i Excel-filen i rätt fält.

En produkt måste finnas på en och endast en rad i Excel-filen. Du kan exportera en produktlista från Bokfri Administration för att se hur Excel-filen kan se ut.

Följande kolumnrubriker kan finnas i Excel-filen:

*Produktnummer *Beskrivning *Försäljningspris (Kan inte vara blank, 0 anges i så fall) *Inköpspris (Kan inte vara blank, 0 anges i så fall) *Enhetsfrakt (Kan inte vara blank, 0 anges i så fall) *Moms *Enhet *Vikt (Kan inte vara blank, 0 anges i så fall) *Volym (Kan inte vara blank, 0 anges i så fall) *Leverantör *Leverantörens artikel nummer *Beställningspunkt (Kan inte vara blank, 0 anges i så fall) *Lagerplats

Nätverk

I Bokfri Administration kan du vara flera användare samtidigt som arbetar med samma företag eller olika företag. För att komma igång och arbeta med nätverk måste ett antal åtgärder vidtas för att det skall fungera att arbeta i nätverk.

Förberedelser
För det första bör man bestämma vilken dator i nätverket som skall vara värddator. Antivirus program kan ställa till problem när man arbetar i nätverk i och med att virusprogrammen vill söka igenom nätverksmappar. På varje dator i nätverket bör man göra ett genomsökningsundantag på nätverksmappen så att inte flera datorer i nätverket söker igenom nätverksmappen samtidigt. På värddatorn måste man köra igång Bokfri Server varje gång innan man börjar arbeta i nätverk, Bokfri Server ligger i programmappen i mappen db. Bokfri Server tar hand om anslutningar till databasmappen, låsning för skrivning och läsning samt postlåsning. I Antivirusprogrammet bör man godkänna programmet Bokfri Server så att programmen i nätverket kan kommunicera med detta program.
Arbeta i nätverk
När förberedelserna är gjorda kan klientdatorerna ansluta sig till Bokfri Server på värddatorn, Bokfri Server ansluter till databasen. I varje klientprogram går man till "Arkiv" och "Nätverksinställningar", där skall ip-adressen till värddatorn eller datornamnet för värddatorn anges så att klientprogrammet kan ansluta till Bokfri Server på värddatorn. Ip-adressen och datornamnet på värddatorn står i huvudfönstret i Bokfri Server på värddatorn. Bokfri Server måste alltid vara igång innan klientprogrammen kan ansluta till databasen och arbeta i nätverk, om Bokfri Server avslutas när klientdatorer är anslutna måste klientprogrammen stängas ned och Bokfri Server startas om.

Online support

Du kan med den här funktionen öppna en webbläsare för att gå till hemsidan för onlinesupport där du kan få nyheter och support för din produkt. För att kunna använda supporten så måste du registrera din produkt, det kan antingen göras via installations programmet eller via supporthemsidan.

Skulle inte hemsidan öppnas så kan du mauellt gå till supportsidan via länken https://www.bokfri.org/

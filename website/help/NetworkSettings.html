Nätverksinställningar

I nätverksinställningar

skriver man in ip-adressen till eller datornamnet för värddatorn som kör Bokfri Server.

Ip-adressen och datornamnet för värddatorn står i huvudfönstret för Bokfri Server då det här programmet är igång.

Bokfri Server måste vara igång för att man skall kunna arbeta i nätverk, om Bokfri Server avslutas när

klientdatorer är anslutna till Bokfri Server måste klientprogrammen stängas ned och Bokfri Server startas om.

Bokfri Server handhar anslutningar till databasen och sköter låsning vid skrivning och läsning samt

postlåsning.

Ta bort postlås

Ibland kan det hända att postlås ligger kvar i Bokfri Server trots att en klientdator

har slutat arbeta med posten. Det här kan hända om ett klientprogram avslutats felaktig när en post är

öppen. Man kan med den här funktionen ta bort alla postlås som ligger och skyddar poster som inte längre

används.

Man bör dock iaktaga försiktighet då man gör det här så att inte någon annan arbetar med poster

för tillfället.
